# Tip Calculator

A simple and clean tip calculator built with HTML, CSS and JavaScript.

## Features
- Calculate tip amount based on percentage
- Split bill amount between multiple people
- Quick-select tip percentage buttons (5%, 10%, 15%, 20%)
- Reset functionality

## How to use
Open index.html in any browser, enter the bill amount, select a tip percentage, and enter the number of people to split the bill.

## Tech used
- HTML5
- CSS3
- JavaScript
